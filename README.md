## Responsive Animated web Application
# 📌 Introduction

1.Designed and implemented floating animated icons and interactive UI elements using JavaScript and CSS to
  create an engaging user experience.
2.Developed dynamic background text spawning effects to enhance visual depth and real-time interactivity.
3.Integrated background music with user-triggered controls, synchronizing audio with UI interactions for an
  immersive interface

---

## ⚙️ Setup Instructions

1. **Save all required assets properly:**

   * Images (photos)
   * Icons
   * Videos (if used)
   * Song (audio file)

2. **Place them in your project folder**
   Example structure:

   ```
   project-folder/
   ├── index.html
   ├── style.css
   ├── script.js
   ├── images/
   ├── icons/
   ├── audio/
   ```

3. **Update file paths in your code**

   * Replace placeholder paths with your actual file names
     Example:

   ```html
   <img src="images/photo1.jpg">
   <audio src="audio/song.mp3">
   ```

4. **Edit content inside the code**

   * Replace sample images with your own
   * Replace text with your custom message
   * Update icons if needed

##  Features

### 🎈 Floating Icons

Icons continuously float across the screen, creating a dynamic and lively feel.

* Adds motion to the UI
* Makes the page feel less static and more interactive

---

### Background Word Spawner

Random romantic words spawn in the background.

* Gives a dreamy aesthetic
* Enhances emotional vibe without cluttering main content

---

###  Interactive Buttons

Buttons respond to user interaction (hover/click effects).

* Improves engagement
* Makes the experience feel responsive instead of dead

---

### Media Integration

* Custom photos can be added
* Optional videos for extra effect
* Everything is user-editable

---

###  Background Song

A song plays in the background to set the mood.

* Can be changed by replacing the audio file
* Adds emotional depth to the project
